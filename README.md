Retail Discount Engine
🔹 Overview

This project is a simple retail discount engine that calculates final prices based on different discount rules. It applies logic to determine how discounts are given depending on product type, quantity, or conditions.

🔹 Features
Applies discounts based on rules
Calculates final price totals
Handles multiple products and conditions
Uses conditional logic and functions
🔹 Technologies Used
JavaScript
HTML
CSS
🔹 How to Run
Open the project folder
Open index.html in your browser
